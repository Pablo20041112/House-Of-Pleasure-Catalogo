📤 Cómo compartir el catálogo

Para que todos vean tus cambios, sigue estos pasos

⚠️ ¿POR QUÉ NO SE ACTUALIZA SOLO?

Los cambios se guardan en tu navegador, no en internet. Cada vez que edites, debes descargar y volver a subir el archivo.

PASO 1 — DESCARGA TU CATÁLOGO

Haz clic en ⬇ Descargar catálogo para obtener el archivo HTML actualizado con todos tus productos.

PASO 2 — SÚBELO GRATIS (elige una opción)

A) Google Drive: Sube el archivo → clic derecho → "Obtener enlace" → "Cualquiera con el enlace" → copia el enlace y compártelo.
B) Netlify Drop (gratis y fácil): Ve a app.netlify.com/drop → arrastra el archivo → obtienes un enlace tipo https://abc123.netlify.app
C) GitHub Pages: Si ya tienes cuenta, sube el archivo a un repositorio y activa GitHub Pages.

PASO 3 — COMPARTE EL ENLACE

Envía el enlace a tus clientes por WhatsApp, Instagram o donde quieras. Cada vez que actualices, solo repite desde el Paso 1.

💡 Consejo: Netlify Drop es la opción más rápida. En menos de 1 minuto tienes tu catálogo en línea.
